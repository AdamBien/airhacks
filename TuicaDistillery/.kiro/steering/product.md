# Tuica Distillery

A Java EE web application for managing a traditional Romanian plum brandy (țuică) distillery catalog.

## Domain
- Catalog management for different tuica varieties (palinca, horinca)
- Quality management and strength validation
- Authentication and principal tracking
- Distillation process monitoring

## Key Concepts
- **Tuica**: The main entity representing a plum brandy with name and alcohol strength
- **Strength**: Alcohol content validation (must exceed minimum threshold)
- **Distillation**: The production process with burner and distillator components
