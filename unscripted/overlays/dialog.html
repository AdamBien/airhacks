Baseline: Widely available · since 2024-09

Replaces: modal libraries, hand-rolled focus traps, backdrop overlays, and scroll locking. Opened here by a declarative invoker command, so the whole modal ships without a script.

The dialog is modal: focus stays inside it, the page behind is inert, and Escape closes it.

Delete workspace?

This modal has a dimmed backdrop, contains keyboard focus, and closes on Escape — all native behavior.

The entire implementation

<button commandfor="demo-dialog" command="show-modal">Open dialog</button>

<dialog id="demo-dialog">
  …content…
  <form method="dialog">
    <button>Cancel</button>
  </form>
</dialog>

form method="dialog" closes the dialog on submit; ::backdrop styles the dimmed layer.
