Exclusive accordion

Baseline: Newly available · name since 2024-09, ::details-content since 2025-09

Replaces: accordion components and their open-state bookkeeping. A shared name makes the group exclusive — the browser closes the previous item; ::details-content animates the disclosure.

Why does opening one item close the other?

The three <details> share name="faq" — exclusivity is native, exactly like radio buttons.

Where is the height animation code?

There is none. ::details-content transitions block-size to auto, with interpolate-size: allow-keywords making the keyword animatable.

Is this accessible?

Keyboard toggling, semantics, and find-in-page support come with the element. The old component versions had to reimplement all three.
