CSS carousel

Baseline: Limited availability · Chrome 135+

Replaces: the entire carousel component category. The markup below is a plain list; the previous/next buttons and the dot markers are CSS-generated pseudo-elements wired to scrolling by the browser.

The entire implementation

.carousel { scroll-marker-group: after; }

.carousel::scroll-button(inline-end) {
  content: "→" / "Next slide";
}

.carousel li::scroll-marker { content: ""; }
.carousel li::scroll-marker:target-current { background: var(--accent); }

Buttons scroll one page and disable themselves at the ends; markers navigate and track the active slide. Keyboard support and accessible names included — by the browser, not by a component.
