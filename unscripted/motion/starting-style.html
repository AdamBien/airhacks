@starting-style

Baseline: Newly available · since 2024-08

Replaces: the classic add-class, wait-for-transitionend, remove-node dance. @starting-style animates the entry from display: none; transition-behavior: allow-discrete keeps the element rendered while its exit plays.

Toggle the toast — it slides and fades in on entry and out on exit, though it toggles between rendered and display: none:

Deployed. Entry and exit are pure CSS.

The entire implementation

.toast {
  opacity: 0;
  translate: 0 1rem;
  transition: opacity .35s, translate .35s,
              display .35s allow-discrete,
              overlay .35s allow-discrete;
}

.toast:popover-open {
  opacity: 1;
  translate: 0 0;

  @starting-style {
    opacity: 0;
    translate: 0 1rem;
  }
}
